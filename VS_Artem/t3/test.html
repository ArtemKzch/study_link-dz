Space

Get details of the SPASE Data Model, which provides terms and syntax for uniform descriptions of Heliophysics resources, including Observatories, Instruments, People, Repositories, and (most centrally) Numerical Data products, An extended set of terms deals with simulations and models. A Dictionary of the terms is provided, along with the XML schema documents used to validate SPASE descriptions. Use of the SPASE base data model along with the simulation extensions are a COSPAR recommendation.
